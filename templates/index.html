Welcome to fediverse.express!

fediverse.express sets up Fediverse instances with minimum knowledge. Right now you can set up a Misskey instance using only your browser. We plan to add more software as this service grows.
fediverse.express is alpha-quality software. While I have tested it extensively and have no reason to believe that it won't work for you, there are things missing and there may be bugs and other issues that prevent it from being stable. If you come across any problems, please let us know.

What is the fediverse?

The fediverse is an interconnected network of social media Web sites that communicate using an open, common protocol. Like Twitter or Facebook, you can write statuses (posts/toots), follow others, like (favorite) and share (boost/repeat) others' posts. The fediverse is not controlled by Big Tech or its algorithms, so it is more democratic and censorship-resistant.

You can read more about the Fediverse here.

Do I need to start my own instance to use the fediverse?

No. Feel free to join some of the instances that already exist. Regardless of which server you join, you can follow nearly anyone on the Fediverse from your instance. However, you are at the mercy of your instance's admin, though it is just as easy to join another instance if you do not like how it is run. Here are some instance recommendations:
  • Mastodon: mastodon.online, mstdn.social, and koyu.space
  • Pleroma: shitposter.club, fedi.absturztau.be, and cdrom.tokyo
  • Misskey: cliq.social

Is this the only way to run my own instance?

No. fediverse.express is not the only service that allows you to create your own instance; how you set up your instance is up to you. If you're technically-savvy and aren't afraid of the command line, you can set up your instance yourself, or you can use a semi-managed service like ours. While fediverse.express was built with the aim that you should have full control over the system that runs your service, masto.host, CofeSpace, and Tribes are other options that trade control for convenience.
